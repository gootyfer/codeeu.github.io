  • Online εκμάθηση JavaScript
  • Online εκμάθηση PHP
  • Online εκμάθηση MySQL
  • Το TEST4U, το οποίο είναι διαθέσιμο στην Ελληνική και Αγγλική γλώσσα, θα σας βοηθήσει:
    - Να εξοικειωθείτε με την επεξεργασία κώδικα. Θα σας ζητηθεί να επεξεργαστείτε τον κώδικα απλών παιχνιδιών, να δείτε το αποτέλεσμα και να το μοιραστείτε με την οικογένειά σας και τους φίλους σας.
    - Θα μάθετε να γράφετε κώδικα. Έχουμε δημιουργήσει μια συλλογή μαθημάτων για αρχαρίους, τα οποία θα σας βοηθήσουν να μάθετε HTML, CSS, JavaScript, PHP και MySQL
